Запускаємо контейнер з відображенням порту контейнера (5000) на порт хоста (5005)

sudo docker run -d -p 5005:5000 rest-apis-flask-python


Монтуємо volume з вихідним кодом в контейнер для розробки, дозволяє оновлювати стан контейнеру в залежності від зміни коду

sudo docker run -d -p 5005:5000 -w /app -v "$(pwd):/app" flask-smorest-api
